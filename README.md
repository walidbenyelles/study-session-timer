# Study Session Timer

Petite web app pour lancer des sessions de révision chronométrées sur une matière donnée.  
L’utilisateur choisit un sujet et une durée en minutes, lance le timer et suit sa progression jusqu’à la fin de la session.

## Fonctionnalités

- Choix du sujet et de la durée de la session (en minutes).
- Affichage d’un compte à rebours au format mm:ss.
- Boutons Start / Pause / Reset pour contrôler la session.
- Message de fin de session et compteur du nombre de sessions terminées aujourd’hui.

## Stack technique

- HTML5  
- CSS3 (mise en page centrée, design type carte)  
- JavaScript (DOM, événements, `setInterval`, logique de timer)

## Objectif du projet

- Pratiquer la manipulation du DOM et la gestion du temps en JavaScript.
- Proposer un petit outil utile pour organiser les sessions de révision.
- Ajouter un projet JavaScript interactif dans mon portfolio front‑end.

## Utilisation

1. Saisir un **subject** (ex : `Math`, `Java`, `English`).
2. Choisir une **duration** en minutes (par défaut 25).
3. Cliquer sur **Start session** pour lancer le compte à rebours.
4. Utiliser **Pause** pour mettre la session en pause, puis **Start session** pour reprendre.
5. Utiliser **Reset** pour remettre le timer à zéro.
